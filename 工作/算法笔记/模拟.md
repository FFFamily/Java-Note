# 题目一：比较版本号

编号：165

思路：模拟遍历整个字符串，对每个数字转换后再比较

代码

```java
class Solution {
    public int compareVersion(String version1, String version2) {
        String[] str1 = version1.split("\\.");
        String[] str2 = version2.split("\\.");
        int len1 = str1.length;
        int len2 = str2.length;
        int index1 = 0,index2 = 0;
        int res = 0;
        while(index1 < len1 || index2 < len2){
            int i1 = 0;
            int i2 = 0;
            if(index1 < len1){
                i1 = Integer.parseInt(str1[index1]);
            }
            if(index2 < len2){
                i2 = Integer.parseInt(str2[index2]);
            }
            if(i1 > i2){
                return 1;
            }else if(i1 < i2){
                return -1;
            }
            index1++;
            index2++;
        }
        return 0;
    }
}
```



# 题目二：斐波那契数列

```
F(0) = 0,   F(1) = 1
F(N) = F(N - 1) + F(N - 2), 其中 N > 1.
```

解答

我自己的答案（但是超时了），相当于暴力解

```java
class Solution {
    public int fib(int n) {
        return dfs(n);
    }
    public int dfs(int n){
        if( n <= 1){
            return n;
        }
        return dfs(n-1) + dfs(n-2);
    }
}
```
